여러분이 <a href="/perl-tutorial">Perl 튜토리알</a>을 여기까지 따라오시는 동안 모든 스크립트들은
여러분이 만든 파일 안에 들어있는 코드의 마지막 줄에 도달했을 때 종료되었습니다.

하지만 더 일찍 실행을 종료하고자 하는 경우가 있습니다.

예를 들어, 사용자에게 나이를 물어보고 13살 미만이라면 스크립트가 종료되는 식으로 말이죠.

=abstract end

<code lang="perl">
use strict;
use warnings;
use 5.010;

print "How old are you? ";
my $age = <STDIN>;
if ($age < 13) {
    print "You are too young for this\n";
    exit;
}

print "Doing some stuff ...\n";
</code>

그저 평범하게 <hl>exit</hl>를 호출하면 됩니다.

<h2>종료 상태값</h2>

여러분이 유닉스/리눅스 셀을 쓴다면, 모든 프로그램이 종료할 때 종료 상태값을 제공하고
그 값은 <hl>$?</hl> 변수에 담기는 것을 알고 계실 겁니다.
펄 스크립트 역시도 <hl>exit()</hl>를 호출할 때 숫자를 인자로 전달함으로써 종료 상태값을
제공할 수 있습니다.

<code lang="perl">
use strict;
use warnings;
use 5.010;

exit 42;
</code>

예를 들어 여기에서는 종료 상태값을 42로 설정했습니다. (기본값은 0입니다.)

<h2>리눅스에서 종료 상태값 확인하기</h2>

유닉스/리눅스 시스템에서는 여러분은 스크립트를 <hl>perl script.pl</hl>처럼 실행했을 것이고, 종료
상태값은 <hl>echo $?</hl>를 써서 검사할 수 있습니다.

<h2>Perl 안에서 종료 상태값 검사하기</h2>

만일, 예를 들어 <a href="https://perlmaven.com/running-external-programs-from-perl">system</a> 함수를 사용하여,
하나의 펄 스크립트 안에서 다른 펄 스크립트를 실행한다면, Perl에도 동일한 <hl>$?</hl> 변수가 있어서
"다른 프로그램"의 종료 상태값을 저장하게 됩니다.

여러분에게 script.pl이라는 코드가 있고 "executor.pl"이라는 또다른 코드가 다음과 같이 되어 있다면:

<code lang="perl">
use strict;
use warnings;
use 5.010;

say system "perl script.pl";
say $?;
say $? >> 8;
</code>

출력은 다음과 같을 것입니다:

<code>
10752
10752
42
</code>

<hl>system</hl> 함수를 호출하면 종료 상태값을 반환하며, 이 값은 또한 Perl의 <hl>$?</hl> 변수에도 저장됩니다.
중요한 것은 이 값은 2바이트 크기이고 실제 종료 상태값은 상위 바이트에 들어간다는 점입니다.
따라서 위에서처럼 42를 다시 얻기 위해서는 비트단위 연산자 <hl>&gt;&gt;</hl>를 사용하여 오른쪽으로
8비트 쉬프트해야 합니다. 위 예제에서 마지막 줄에서 볼 수 있습니다.
