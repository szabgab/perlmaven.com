How to read the documentation of Perl using perldoc or in the browser.


=abstract end

<slidecast file="beginner-perl/documentation" youtube="dNvDSO_wKBE" />

<code>
perldoc perl
perldoc perlsyn
perldoc -f print

perldoc -q sort
</code>

You can access the documentation of Perl via the <a href="http://perldoc.perl.org/">perldoc.perl.org</a> site,
but today I prefer to read it on <a href="https://metacpan.org/">MetaCPAN</a>. In order to make it easier
to find various strange operators of Perl, you can search for them in the menu-bar of the
Perl-Maven site.
