Finding the right module on CPAN can be a daunting task. For many problems there are tens or even hundreds of solutions with
different focus, different API, and different level of quality. <a href="https://metacpan.org/pod/Task::Kensho">Task::Kensho</a>
is a list of recommended modules for certain common tasks.

=abstract end

<screencast file="metacpan-task-kensho" youtube="e3ZqUBCVjIQ" />

<a href="https://metacpan.org/pod/Task::Kensho">Task::Kensho</a> itself is divided into several subjects:

<ul>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Async">Task::Kensho::Async</a> for asynchronous programming.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::CLI">Task::Kensho::CLI</a> providing useful Command-line Tools.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Config">Task::Kensho::Config</a> for configuration of applications.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::DBDev">Task::Kensho::DBDev</a> for Database Development.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Dates">Task::Kensho::Dates</a> manipulating dates and timestamps.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Email">Task::Kensho::Email</a> for validating email address and sending e-mails with attachments.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::ExcelCSV">Task::Kensho::ExcelCSV</a> to read and write Excel and/or CSV files.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Exceptions">Task::Kensho::Exceptions</a> to catch exceptions and to automatically generate exceptions.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Hackery">Task::Kensho::Hackery</a> for hacking?</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Logging">Task::Kensho::Logging</a> for logging in modules and applications.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::ModuleDev">Task::Kensho::ModuleDev</a> helping to write better quality code, helping to package modules.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::OOP">Task::Kensho::OOP</a> Object Oriented Programming</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Scalability">Task::Kensho::Scalability</a></li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Testing">Task::Kensho::Testing</a></li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::Toolchain">Task::Kensho::Toolchain</a> to make it easier to work with CPAN.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::WebCrawling">Task::Kensho::WebCrawling</a> - accessing web site from your Perl script.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::WebDev">Task::Kensho::WebDev</a> tools and frameworks for Web Development.</li>
  <li><a href="https://metacpan.org/pod/Task::Kensho::XML">Task::Kensho::XML</a> for processing XML and creating XML files.</li>
</ul>
